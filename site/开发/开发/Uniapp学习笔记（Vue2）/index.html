    此配置需要在项目根目录的pages.json中进行。

    温馨提示

    1. uni-app为了调试性能的原因，修改easycom规则不会实时生效，配置完后，您需要重启HX或者重新编译项目才能正常使用uView的功能。
    2. 请确保您的pages.json中只有一个easycom字段，否则请自行合并多个引入规则。
    3. 如果您是通过uni_modules形式引入uView，可以忽略此配置

    import Vue from 'vue'
import Vuex from 'vuex'

Vue.use(Vuex) // vue的插件机制

const store = new Vuex.Store({
    //数据
  state: {
    counter: 0
  },
  //get set
  mutations: {
    incrementCounterMutations(state) {
      state.counter++;
    },
    setCounterMutations(state, payload) {
      state.counter = payload;
    }
  },
  //发送请求，commit提交，调用mutactions方法来修改state
  actions:{
    incrementCounterActions(context) {
        context.commit('incrementCounterMutations');
    },
    setCounterActions(context, payload) {
        context.commit('setCounterMutations', payload);
    }
  }
});

export default store;
